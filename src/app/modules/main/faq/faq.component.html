Frequently Asked Questions

Do you have any questions? Please check the document below. You might find the answer. If you do not,

How to install on Windows

  • Step One : Download the windows installer from intellifriend.com and run the exe file by clicking on Run



  • Step Two : Read the License and click 'I Agree'



  • Step Three : Check user option and click next



  • Step Four : Browse the installation path and click install



  • Step Five : Wait for a while to finish installation and then click Finish



How to install on Ubuntu

  • Step One : Download the ubuntu.zip from the website intellifriend.com. Extract the file then you will see a bash file named setup.sh.Open the terminal and navigate to the extracted folder. Run the following two commands.

    • chmod +x setup.sh
    • ./setup.sh


How to install on MacOS

  • Step One : Download the Mac Installer from intellifriend.com

  • Step Two : Open the dmg and drag the app to Applications. Open intelliFriend from Applications



  • Step Three : The following message will pop up just click open and you will be redirected to general tab in System preference



  • Step Four : First, unlock for editing by clicking "Click the lock to make changes.".



  • Step Five : Enter your username and password into the prompt that appears and click Unlock.



  • Step Six : Now, click "Open Anyway."



  • Step Seven : App will launch, then go to system preferences in the privacy tab.





  • Step Eight : Give permission to intellifriend.
